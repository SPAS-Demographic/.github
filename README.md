## What SPAS does

We compile, harmonize, and publish sub-national population data by age and sex worldwide, integrating data from multiple international demographic programs with geographic boundary files to enable spatially explicit analysis of population composition at fine spatial scales.

## Where to find the data

- 📦 **[AgeSex-Pop](https://github.com/SPAS-Demographic/AgeSex-Pop)** — main data collection (GeoPackage). *Currently private; public release planned after peer review.*
